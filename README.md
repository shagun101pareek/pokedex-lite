# 🧩 Pokedex Lite

A modern Pokedex web application built using **React, TypeScript, and TailwindCSS**.  
It allows users to browse Pokémon, search by name, filter by type, and mark favorites.

---

## 🚀 Features

- 🔍 Search Pokémon by name  
- 📄 Pagination (Next / Previous navigation)  
- ❤️ Add / Remove Favorites  
- 🎛️ Filter Pokémon by Type  
- ⭐ Favorites filtering mode  
- ⏳ Loading spinner  
- 🎨 Type-based card styling  
- 📱 Responsive UI  

---

## 🛠️ Tech Stack

- React
- TypeScript
- TailwindCSS
- Vite
- PokeAPI
